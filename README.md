# atd-product

A website about ATD's Data & Technology Services team.

## Get it running

Local development requires `git` and `npm`.

Clone this repo

```
$ git clone git@github.com:cityofaustin/atd-product.git
```

Install packages

```
$ npm install
```

Start server

```
$ npm run dev
```
